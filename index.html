Temperature Sensor Signal Conditioning PCB

Altium PCB Design Analog

Winter, 2025

View on GitHub →

Analog front-end for a TMP36 temperature sensor: 5 V input regulated to 3.3 V through an LDO (chosen over a buck converter for lower noise), with 0.1 µF and 1 µF shunt decoupling. Signal path is LDO → TMP36 → non-inverting gain stage → RC low-pass filter (~10 Hz) → unity-gain buffer → ADC/microcontroller output, using rail-to-rail, single-supply 3.3 V op-amps.

  • Gain stage scales mV sensor output with high input impedance so the TMP36 is not loaded
  • ~10 Hz RC filter after gain attenuates ripple and noise on the slow temperature signal
  • Unity-gain buffer isolates the filter from the ADC with low output impedance
  • Ground plane and tight decoupling loops for clean analog performance
